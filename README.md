# このリポジトリについて  
このリポジトリは仙台高専広瀬の学生チームによるETロボコン2017用のリポジトリです。  
出場部門はデベロッパ部門アドバンストクラスで、今年度が初出場です。  

## チーム内の方へ  
チームslackに更新が通知されます。  通知がありましたらgit pullなどで常にブランチを最新のバージョンにしてください。

また開発環境のバージョンが人それぞれ違うようなので、今のところは管理対象に含めないでください。  
(.gitignoreで無視するようにはしてありますが) 

プライベートリポジトリではないので、管理対象に含めるファイルには気を付けてください。  

## チーム外の方へ  
参考にされるのは構いませんが、pull requestは多分受け付けないと思います。
